# Frontend

Next.js App Router frontend for the thesis defense hackathon demo.

## Commands

```bash
pnpm install
pnpm dev
pnpm typecheck
pnpm build
pnpm start
```

## Routes

- `/` animated landing page
- `/login`
- `/dashboard`
- `/admin`
- `/admin/users`
- `/admin/groups`
- `/admin/schedules`
- `/admin/theses`
- `/admin/statistics`
- `/student`
- `/student/thesis`
- `/student/files`
- `/student/scores`
- `/student/schedule`
- `/student/critiques`
- `/teacher`
- `/teacher/mentor-students`
- `/teacher/critique-groups`
- `/teacher/scoring`
- `/teacher/reviews`

Set `NEXT_PUBLIC_API_URL=http://localhost:4001/api` for local development.
